## Локальная разработка

```bash
npm install
npm run dev
```

Откройте http://localhost:3022

## Статическая сборка (как на GitHub)

```bash
npm run build
# env: SKIP_VERIFY_EXPORT=1 BASE_PATH=/GE_Vol.2
npm run start
```

Папка `out/` — готовый статический сайт для любого хостинга.
